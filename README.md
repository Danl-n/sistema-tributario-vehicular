# Sistema Tributario Vehicular

Proyecto basado en arquitectura de microservicios para consultar información tributaria y vehicular mediante integraciones externas con el SRI y ANT.

## Componentes

* Frontend Web
* API Gateway
* MS Contribuyente
* MS Vehicular
* MS ANT
* Redis Cache

## Tecnologías

* Spring Boot
* React
* Redis
* Docker

## Patrón Implementado

* Cache Aside Pattern
